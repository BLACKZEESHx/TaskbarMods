Start Killer v6.0 portable

Removes the start menu and (optionally blocks the start menu). Run the startkiller.exe to remove the start button. Exit the Start Killer to return the start button back.

Package folders:
32 - contains Start Killer 32-bit version. Use this version for 32-bit Windows.
64 - contains Start Killer 64-bit version. Use this version for 64-bit Windows.

Command line parameters:

Exit Start Killer:
startkiller.exe exit

Open Start Killer properties:
startkiller.exe prop
